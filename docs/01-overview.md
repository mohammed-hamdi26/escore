# نظرة عامة على المشروع - E-Score Dashboard

## ما هو المشروع؟

E-Score هو لوحة تحكم إدارية لإدارة محتوى الرياضات الإلكترونية (Esports). يتيح للمسؤولين إدارة:
- اللاعبين والفرق
- المباريات والبطولات
- الألعاب
- الأخبار
- الانتقالات
- المستخدمين
- الدعم الفني
- إعدادات التطبيق

## التقنيات المستخدمة

### Framework الأساسي
- **Next.js 15.5.4** - مع App Router
- **React 19.1.0** - أحدث إصدار
- **Turbopack** - للبناء السريع

### واجهة المستخدم
- **Tailwind CSS 4** - للتنسيق
- **Shadcn/ui** - مكونات UI جاهزة
- **Radix UI** - مكونات أساسية
- **Lucide React** - أيقونات

### إدارة البيانات
- **Axios** - لطلبات HTTP
- **Formik + Yup** - للنماذج والتحقق
- **@tanstack/react-table** - للجداول

### الترجمة والثيمات
- **next-intl** - للترجمة (العربية/الإنجليزية)
- **next-themes** - للثيمات (فاتح/داكن)

### أدوات أخرى
- **date-fns** - للتواريخ
- **react-hot-toast** - للإشعارات
- **cmdk** - للبحث

## بنية المجلدات الرئيسية

```
escore-1/
├── app/
│   └── [locale]/              # المسارات حسب اللغة
│       ├── _Lib/              # API والدوال المساعدة
│       ├── dashboard/         # صفحات لوحة التحكم
│       ├── login/             # صفحة تسجيل الدخول
│       └── register/          # صفحات التسجيل
├── components/
│   ├── ui/                    # مكونات Shadcn
│   ├── ui app/                # مكونات مخصصة
│   ├── dashboard/             # مكونات Dashboard
│   ├── icons/                 # أيقونات SVG
│   └── [feature]/             # مكونات لكل ميزة
├── lib/                       # أدوات مساعدة
├── i18n/                      # إعدادات الترجمة
├── messages/                  # ملفات الترجمة
└── public/                    # ملفات ثابتة
```

## الأوامر الأساسية

```bash
# تشغيل بيئة التطوير
npm run dev

# بناء المشروع
npm run build

# تشغيل الإنتاج
npm start

# فحص الكود
npm run lint
```

## المتغيرات البيئية

## الميزات الرئيسية

1. **إدارة اللاعبين** - إضافة/تعديل/حذف اللاعبين
2. **إدارة الفرق** - إضافة/تعديل/حذف الفرق
3. **إدارة المباريات** - جدولة وإدارة المباريات
4. **إدارة البطولات** - إنشاء وإدارة البطولات
5. **إدارة الألعاب** - إضافة الألعاب المدعومة
6. **إدارة الأخبار** - نشر وتحرير الأخبار
7. **إدارة الانتقالات** - تسجيل انتقالات اللاعبين
8. **إدارة المستخدمين** - إدارة حسابات المستخدمين
9. **الدعم الفني** - الرد على تذاكر الدعم
10. **الإعدادات** - اللغات، الثيمات، المحتوى
